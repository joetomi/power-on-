# Router Status

واجهة متجاوبة لمراقبة توفر الراوتر. تنفذ فحصًا خلفيًا كل 5 ثوانٍ، وتعرض حالة الطاقة وزمن استجابة HTTP وجودته ووقت آخر فحص، مع مؤشر بصري أخضر عند توفر الراوتر أو أحمر عند تعذر الوصول إليه.

## التشغيل محليًا

```bash
npm install
npm run dev
```

ثم افتح `http://localhost:3000`.

## التحقق

```bash
npm run typecheck
npm run build
```

## النشر على Vercel

اربط المستودع بمشروع Vercel أو نفّذ `vercel` من مجلد المشروع. لا يتطلب المشروع متغيرات بيئة أو بيانات دخول للراوتر.
