# Router Status

واجهة بسيطة لمراقبة توفر الراوتر. تنفذ الواجهة فحصًا عبر مسار خلفي كل 10 ثوانٍ، وتعرض دائرة خضراء عند توفره أو حمراء عند تعذر الوصول إليه.

## التشغيل محليًا

```bash
npm install
npm run dev
```

ثم افتح `http://localhost:3000`.

## التحقق

```bash
npm run typecheck
npm run build
```

## النشر على Vercel

اربط المستودع بمشروع Vercel أو نفّذ `vercel` من مجلد المشروع. لا يتطلب المشروع متغيرات بيئة أو بيانات دخول للراوتر.
